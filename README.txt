使用短码方式，添加网易云音乐官方播放器。

== Description ==

直接调用网易云音乐官方外链播放器，防止因api失效导致无法播放。

提示：对版权保护音乐无效

== Installation ==

1. 上传文件到 `/wp-content/plugins/official-player-of-netease-music` 目录，或后台搜索official player of netease music
2. 在WordPress插件管理中激活插件
3. 添加短码[nm-song]歌曲id[/nm-song]到文章或页面中。


== Frequently Asked Questions ==

= 歌曲是自动播放的吗？ =

目前只支持手动播放，后续更新会支持。

= 支持播放版权保护的音乐吗？ =

不支持
